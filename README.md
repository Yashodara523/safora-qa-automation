# Safora QA – Contact Form Automation

**Framework:** Playwright (Node.js)
**Tested URL:** https://safora.se/en/contact.html

## How to run

1. Install Node.js from https://nodejs.org
2. Open terminal, go to project folder
3. Run: npm install @playwright/test
4. Run: npx playwright install chromium
5. Run: npx playwright test safora_contact_form.spec.js --headed
